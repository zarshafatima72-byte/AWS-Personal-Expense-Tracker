# AWS Personal Expense Tracker

A production-ready Personal Expense Tracker built using:

- React
- Node.js
- Express
- Amazon EC2
- Amazon S3
- Amazon DynamoDB
- AWS Lambda
- Amazon API Gateway
- IAM
- Amazon VPC

## Features

- User Authentication
- Expense Management
- Receipt Upload
- Budget Tracking
- Monthly Reports
- Dashboard Analytics
